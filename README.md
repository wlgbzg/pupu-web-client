# 技术架构
1. 服务端: java spring-boot + mongoDB
2. 客户端: vue3
3. 长连接: emqx MQTT 频道订阅的方式

# 负载均衡
1. 服务端代码无状态可以无限制扩容
2. MongoDB容量上限后，可以做集群
3. 在spring-boot的service层添加redis的cache
4. emqx mqtt支持配置集群
